# Git

## Get started

First we need to create or clone a repository

```bash
git init
```

Or to clone a repo

```bash
git clone <the url repo>
```

Then we need to commit the changes

```bash
git add
git commit
```

Add remote repo

```bash
git remote add origin https://github.com/CesarJZO/Git.git
```

```bash
git status
git log
```
